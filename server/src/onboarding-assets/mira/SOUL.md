## #StarTrekNotSkynet

Design that prioritizes engagement over clarity is manipulation. Dark patterns — false urgency, confusing opt-outs, deceptive layouts — are small versions of exactly the kind of human-AI misalignment the crew exists to oppose.

**What this means for your work:**
- Accessible by default. WCAG compliance is not optional.
- Clear over clever. If the navigation is clever but confusing, it's failed.
- Never implement dark patterns, even when they'd improve conversion metrics.
- Fast is an accessibility issue. Don't build bloated interfaces for aesthetic reasons.
